# Features
- Offline, the characters data is in js file present in ```AnkiDroid/collection.media/``` folder
- Show or hide Simplified, Traditional characters, Pinyin or Meaning
- HSK 1
- New design

# Downloads
[xiehanzi 2.0 HSK 1](https://github.com/infinyte7/Anki-xiehanzi/blob/master/Version%202.0/xiehanzi%202.0.apkg?raw=true)

# Setup
1. Download this GitHub repo [Hanzi writer data in javascript](https://github.com/infinyte7/hanzi-writer-data-in-javascript)

<img src="https://raw.githubusercontent.com/infinyte7/Anki-xiehanzi/master/Version%202.0/images/download_repo.gif" height="640" width="360"></img>

2. Copy ```char_data_js``` folder to ```AnkiDroid/collection.media```

<img src="https://raw.githubusercontent.com/infinyte7/Anki-xiehanzi/master/Version%202.0/images/exctract_copy.gif" height="640" width="360"></img>

3. Import downloaded xiehanzi 2.0 deck

<img src="https://raw.githubusercontent.com/infinyte7/Anki-xiehanzi/master/Version%202.0/images/download_import.gif" height="640" width="360"></img>

3. Open AnkiDroid & turn on fullscreen

<img src="https://raw.githubusercontent.com/infinyte7/Anki-xiehanzi/master/Version%202.0/images/make_fullscreen.gif" height="640" width="360"></img>

4. Select the ```xiehanzi 2.0``` and start writing
